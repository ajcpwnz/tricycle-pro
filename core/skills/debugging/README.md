# Debugging Skill

Structured reproduce-isolate-trace-fix-verify debugging workflow.

## Installation

This skill is installed automatically by `tricycle init`. It can be invoked via `/debugging` in Claude Code.

## Usage

```
/debugging
```

Guides the agent through systematic root-cause analysis rather than trial-and-error debugging.

## Customization

Edit `.claude/skills/debugging/SKILL.md` to adjust the debugging workflow for your project's debugging tools and patterns.
