# TDD Skill

Red-Green-Refactor test-driven development workflow.

## Installation

This skill is installed automatically by `tricycle init`. It can be invoked via `/tdd` in Claude Code.

## Usage

```
/tdd
```

Guides the agent through writing failing tests first, then implementing the minimum code to pass, then refactoring.

## Customization

Edit `.claude/skills/tdd/SKILL.md` to adjust the TDD workflow for your project's testing framework and conventions.
