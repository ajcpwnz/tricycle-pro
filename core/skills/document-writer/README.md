# Document Writer Skill

Document generation for DOCX, PDF, PPTX, and other formatted documents.

## Installation

This skill is installed automatically by `tricycle init`. It can be invoked via `/document-writer` in Claude Code.

## Usage

```
/document-writer
```

Guides the agent through creating structured, professional documents.

## Customization

Edit `.claude/skills/document-writer/SKILL.md` to adjust the document creation workflow for your organization's templates and conventions.
