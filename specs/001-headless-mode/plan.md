**Branch**: `001-headless-mode` | **Date**: 2026-03-24 | **Spec**: [spec.md](spec.md)
**Input**: Feature specification from `/specs/001-headless-mode/spec.md`

## Summary

Add a `/trc.headless <prompt>` slash command that orchestrates the
full specify → plan → tasks → implement chain in a single invocation.
The command is a markdown prompt file (like all other trc commands)
that instructs Claude to execute each phase sequentially, auto-resolve
non-critical decisions, and pause only for critical clarifications,
destructive actions, or push approval.
